Confirm Sale Order, pikcing or invoice from the same view (Sale view)

Odoo Sales Order requires many actions, such as converting a quotation to an order, creating an invoice, validating an invoice, registering payment for an invoice. Normally, one need to perform these actions manually to complete a Sales Order. Auto confirm picking invoice from sale app provides configuration workflow to allow all or some of these specific processes and action. Once you install the app, you will be able to confrim and print sale, picking and invoice from the same view.
